# NETGHOST-01: Enumeration Fundamentals

NETGHOST-01 is a beginner-friendly cybersecurity lab focused on network reconnaissance and enumeration.

Participants take the role of a junior penetration tester assessing the internal network of Aegis Corp.

The objective is to analyze simulated outputs from common enumeration tools and extract hidden flags.

## Skills Covered

- Host discovery
- Port scanning
- Service enumeration
- Banner grabbing
- OS fingerprinting
- SMB enumeration

## Lab Structure

1. Ghost Hunt – Host discovery
2. Open Doors – HTTP header analysis
3. FTP Fingerprint – FTP banner enumeration
4. OS Oracle – Operating system fingerprinting
5. Service Census – SMB enumeration

## Tools Referenced

- nmap
- curl
- netcat
- ftp
- smb-os-discovery
